# v3design

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
node builder.js
```

### 静态页面查看方法
```
点击页面生成页面后，
项目下会产生dist文件夹，
需要手动在dist下手动启动一个server, 推荐 http-server。
```

### Compiles and minifies for production
```
npm run build
注: build出来的不是项目本身，而是拖拽后的页面。
本项目前demo阶段， 只提供思路，暂没有拆分前后端以及npm包。
```
